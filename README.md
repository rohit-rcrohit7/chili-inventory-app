# CHILI Inventory Manager (Streamlit + Google Sheets)

This Streamlit app provides a live, multi-user inventory management system for environmental monitoring equipment. It syncs directly with Google Sheets for persistent, collaborative data access.

---

## 🚀 Features

- Add, check-in, and check-out items
- Auto-creates and syncs with a Google Sheet
- Tracks:
  - Item type, ID, serial, location
  - Checkout status, dates, responsible person
  - **New:** "Checked Out Location" field
- Full audit log for transparency
- Download CSVs for backup or analysis
- Visual dashboard with charts
